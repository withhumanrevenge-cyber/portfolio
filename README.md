## 🚀 Technical Architecture
This portfolio is a **Living Document**. It operates as an autonomous engine that syncs directly with my GitHub ecosystem.

### Core Stack
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Runtime**: [React 19](https://react.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Visuals**: [Three.js](https://threejs.org/) & [Lucide](https://lucide.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom design system.

## 🛠️ Engineered Features

### 1. Autonomous Project Sync
I hate manual updates. My portfolio implements a custom synchronization layer (`src/lib/github.ts`) that:
- Periodically fetches real-time repository data.
- Filters for projects tagged with `portfolio`.
- Automatically generates high-fidelity case studies via a `case_study.json` schema found in my repo roots.
- Updates the work section in real-time—zero manual intervention required.

### 2. 3D Interaction Design
The centerpiece is a hardware-accelerated **3D Photo Carousel**. It utilizes custom perspective transforms and physics-based drag gestures to create a cinematic browsing experience for my featured work.

### 3. Radical Efficiency
Optimized for Core Web Vitals with:
- **Incremental Static Regeneration (ISR)** for blazingly fast load times.
- **Main-thread isolation** for heavy 3D logic to ensure sub-millisecond interaction latency.
- **Clean Code Architecture**: A modular, maintainable codebase stripped of all unnecessary boilerplate.
